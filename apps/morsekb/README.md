# Morse Keyboard

A Morse code keyboard for text input

written by: [Spiegie](https://github.com/Spiegie)

# Usage

- press the upper corners for dash and dots.
- press lower left corner (ret) to commit a character.
- press lower left corner (ret) 3 times to create a space.
- press lower right corner (del) to delete a character. if no morse signals are present, the output last character from output text is removed
- swipe to the right to submit your text
- swipe to the left to cancel the textinput
- swipe up to show a selection of special characters
